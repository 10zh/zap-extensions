Options Global Alert Filters

This screen allows you to configure Global Alert Filters which allow you to automatically override the risk levels of any alerts raised by the active and passive scan rules. Unlike Context Alert Filters they apply to all alerts raised, not just those raised in a specific context.

It shows a list of all of the Global Alert Filters along with buttons for adding, removing, and deleting them.

Note: The alert filters will only be applied to new alerts, not to any alerts that were already present when the filter was created.

The 'add' and 'modify' dialogs have the same fields as Context Alert Filters.
