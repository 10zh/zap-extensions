Zest

Zest egy kísérleti speciális parancsnyelv (úgy is ismert, mint egy domain-specifikus nyelv) által kifejlesztett, a Mozilla biztonsági csapat célja, hogy használható web-orientált biztonsági eszközök.

It is included by default with ZAP.

Creating Zest scripts

Vannak különböző módon lehet létrehozni Lelkesedés szkriptek:

Rekord egy új Lendülettel script Gombra

  • Nyomja meg a 'Rekord egy új Lendülettel script' gombra a fő eszköztáron
  • Type in a suitable name for your script in the 'Add a Zest Script' dialog
  • Válassza ki az előtag azt szeretné, hogy felvétel iránti kérelmet, vagy hagyja üresen, hogy rögzítse az összes kérelem
  • Press the 'Save' button
  • A 'Rekord egy új Lendülettel script' gombra marad, préselt, változni, hogy a Felvétel egy új Lendülettel script' s show, egy vörös ikon.
Az új Lendülettel script lesz látható a Szkriptek lap piros 'felvétel' ikonra.
Bármilyen kérés, hogy alatta a megadott prefix-lesz hozzá, hogy a forgatókönyvet.
Press the 'Recording a new Zest script' again to stop recording the requests.
Vegye figyelembe, hogy csak rekord 'Önálló' Lelkesedés szkriptek ezen a módon. Ha szeretné, hogy hozzon létre más típusú Lelkesedés forgatókönyvet kell használni egy másik mechanizmus.

Akkor is helyes, kattintson a minden Önálló Lelkesedés parancsfájl használata a felvétel megkezdéséhez", valamint a "felvétel leállításához' gombot.

New Script Button

  • Keresse meg a Szkriptek fa lap
  • Press the 'New Script...' button
  • Írja be a megfelelő nevet a forgatókönyvet az Új Script' párbeszédpanel
  • Válassza ki a forgatókönyv típusú (lásd a Szkriptek add-on segíteni oldalt a részletekért)
  • Válassza ki a Legjobb forgatókönyv motor
  • Válassza ki az egyik a sablonok (amennyiben releváns)
  • Nyomja meg a "Mentés" gombra
Bármilyen típusú Lelkesedés forgatókönyvet is létre lehet hozni erre.

Jobb kattintás egy Lendülettel sablon

  • Navigate to the Scripts tree tab
  • Bontsa ki a 'Sablonok' csomópont megtalálni a sablont szeretnéd használni
  • Kattintson jobb gombbal a sablonnak, majd válassza az Új Script...'
  • Press the 'Save' button
Any type of Zest script can be created this way.

Right clicking requests

  • Keresse meg minden lapot, ami megmutatja, kéréseket, mint például a Történelem lap
  • Select one or more requests
  • Right click on them
  • Válassza ki a Hozzáadás a Lelkesedés Script' menü, amely lehetővé teszi, hogy válasszon ki egy meglévő Önálló script, vagy hozzon létre egy újat
Vegye figyelembe, hogy csak hozzá kéréssel, hogy 'Egyedül' Lelkesedés szkripteket.

Plug-n-Hack

Ha a legfrissebb Firefox akkor hozzon létre Lelkesedés szkriptek belül a böngésző.
  • Press the 'Plug-n-Hack' button on the ZAP 'Quick Start' tab
  • Telepítse a Plug-n-Hack Firefox Add-on, majd fogadja el a párbeszédpaneleken
  • Press 'Shift F2' in Firefox to access the Developer Toolbar
  • Típus 'zap rekord a globális' a felvétel megkezdéséhez egy új Lendülettel forgatókönyv
  • Bármilyen kérés, hogy át ZAP lesz hozzá, hogy a forgatókönyv
  • Type 'zap record off global' to stop recording the script
Note that you can only record 'Stand Alone' Zest scripts in this way. If you want to create other types of Zest script you must use another mechanism.

Szerkesztés Lelkesedés parancsfájlok

Zest forgatókönyv szerkesztett grafikusan a Szkriptek fa lapon.
Minden mondat egy csomópont a fa -, kattintson duplán a csomópont szerkesztése a nyilatkozatot a tulajdonságok parancsra.
Felveheti, mozgás, majd távolítsa el kimutatások keresztül jobb kattintás a Lelkesedés csomópontok.
Azt is hozzá kéri, hogy 'egyedül' Lelkesedés parancsfájlok által kattintson a kéréseket a többi lapok.
There are also some right click options available when you select text in the Request or Response tabs.

Zest includes a set of 'built in' variables as well as allowing you to declare your own.
A right click menu is provided (where relevant) in the edit dialogs to allow you to paste in any of the available variable names.
