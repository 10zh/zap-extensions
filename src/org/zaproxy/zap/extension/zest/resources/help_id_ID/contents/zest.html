Zest

Semangat adalah eksperimental berupa bahasa scripting (juga dikenal sebagai domain-spesifik bahasa) dikembangkan oleh Mozilla tim keamanan dan dimaksudkan untuk digunakan di web berorientasi alat-alat keamanan.

Ini termasuk secara default dengan ZAP.

Membuat skrip Zest

Ada berbagai cara untuk membuat Semangat script:

Catat sebuah skrip Tombol baru Zest

  • Tekan tombol 'Record a new Zest script' tombol pada toolbar utama
  • Ketik nama yang cocok untuk naskah anda di 'Add a Zest Script' dialog
  • Pilih awalan anda ingin merekam permintaan, atau biarkan kosong untuk merekam semua permintaan
  • Tekan tombol 'save'
  • 'Record a new Zest script' tombol akan tetap ditekan, perubahan untuk 'Recording a new Zest script' dan menunjukkan ikon merah.
Baru Semangat script akan ditampilkan di Script tab dengan merah 'recording' icon.
Setiap permintaan yang anda buat di bawah yang ditentukan awalan akan ditambahkan ke script.
Tekan tombol 'Recording a new Zest script' lagi untuk berhenti merekam permintaan.
Perhatikan bahwa anda hanya dapat merekam 'Stand Alone' Semangat script dengan cara ini. Jika anda ingin membuat jenis lain dari Semangat script anda harus menggunakan mekanisme lain.

Anda juga dapat klik kanan setiap Stand Alone Semangat script dan menggunakan 'Start recording' dan 'Stop recording' tombol.

Tombol Skrip Baru

  • Navigasi ke tab pohon Skrip
  • Tekan tombol 'New Script...'
  • Ketik nama yang cocok untuk anda script di 'New Script' dialog
  • Pilih jenis script (lihat Script add-on halaman bantuan untuk informasi lebih lanjut)
  • Pilih mesin skrip Zest
  • Pilih salah satu template (jika relevan)
  • Tekan tombol 'Save'
Setiap jenis dari skrip Zest bisa dibuat memlalui cara ini.

Klik kanan template Zest

  • Navigate to the Scripts tree tab
  • Perluas node 'Templates' dan cari sebuah template yang anda ingin gunakan
  • Klik kanan pada template dan pilih 'New Script...'
  • Tekan tombol 'Save'
Setiap jenis dari skrip Zest bisa dibuat melalui cara ini.

Klik kanan permintaan

  • Arahkan ke tab mana saja yang menampilkan permintaan, seperti tab riwayat
  • Pilih satu permintaan atau lebih
  • Klik kanan pada mereka
  • Pilih menu 'Tambah ke Zest Script' yang akan memungkinkan anda untuk memilih Stand Alone Script yang ada atau buat yang baru
Perhatikan itu anda hanya dapat menambah permintaan ke 'Stand Alone' zest Script.

Plug-n-Hack

Jika anda menggunakan Firefox versi baru maka anda bisa membuat Zest Script dari dalam browser anda.
  • Tekan tombol 'Plug-n-Hack' pada tab 'Quick Start' ZAP
  • Memasang Add-on Plug-n-Hack Firefox dan terima semua dialog
  • Tekan 'Shift F2' dalam Firefox untuk mengakses Toolbal pengembang
  • Tuliskan 'zap record on global' untuk mulai merekam Zest script baru
  • Setiap permintaan yang anda buat melalui Zap akan ditambahkan ke script
  • Ketik 'zap record off global' untuk menghentikan rekaman script
Perhatikan itu anda hanya dapat merekam 'Stand Alone' Zest script dengan cara ini. jika anda ingin membuat tipe Zest script lain anda harus menggunakan mekanisme lain.

Mengubah Zest script

Zest script diubah secara grafik di tab pohon script.
Setiap pernyataan adalah simpul di pohon - klik ganda simpul untuk mengubah properti pernyataan.
Anda dapat menambah, memindahkan dan menghapus pernyataan via klik kanan simpul Zest.
Anda juga dapat menambah permintaan ke 'Stand alone' Zest script dengan mengklik kanan permintaan dalam salah satu tab lain.
Juga ada beberapa pilihan klik kanan yang tersedia saat anda pilih teks di tab Permintaan atau Respon.

Zest mencakup seperangkat variabel 'built in' serta memungkinkan anda untuk menyatakan milik anda sendiri.
Klik kanan menu yang disediakan (jika relevan) dalam mengedit dialog untuk memungkinkan anda untuk paste di manapun tersedia nama-nama variabel.
