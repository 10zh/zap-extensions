Export Report

Laporan Ekspor modul yang memungkinkan pengguna untuk menyesuaikan konten dan ekspor dalam format yang diinginkan.
  • Dibuat oleh JordanGS.
  • Minimum Versi Yang Didukung: mingguan rilis ZAP_D-2016-09-05
  • Project on GitHub. You can also find a .zip of the minimum supported release in the zap-download folder.
  • Didukung dan dimasukkan kedalam resmi OWASP Zed Attack Proxy Jenkins Plugin.

Daftar Isi

  • ZAP UI
  • Baris Perintah
  • API panggilan

ZAP UI

Source Info

Pengguna masuk dan secara otomatis mengambil data yang relevan untuk laporan. Arahkan kursor ke masing-masing bidang dalam ekstensi untuk tool tip.
  • Judul - Default untuk Sesi Nama
  • By
  • For
  • Upload Scan - Pengguna memasukkan tanggal AScan, default untuk saat ini tanggal-waktu
  • Tanggal Laporan - Default untuk saat ini tanggal-waktu
  • Versi Scan - Placeholder untuk masa depan
  • Laporan Versi - Default untuk versi saat ini dari ZAP alat

UI

File Chooser Dialog

Dialog hanya menampilkan folder dan file yang diterima. Yang 'Tipe file daftar drop-down akan filter untuk hanya menampilkan folder dan file-file dengan ekstensi tertentu.

Currently Available

  • ASCII 1.0 Strict Compliant XHTML Files (.xhtml )
  • XML Files (.xml )
  • JSON Files (.json )

Future Milestone

  • Bootstrap HTML Files(.bootstrap.html )
  • PDF Documents (.pdf )
  • Google Documents (.doc )

Command Line

Ekstensi dapat dijalankan dari baris perintah serta dan membutuhkan argumen berikut yang harus dilalui untuk menghasilkan laporan.

OWASP ZAP Command Line Options

  • -session: Opens the given session after starting ZAP
  • -cmd: Runs ZAP 'inline', ie without starting the UI or a daemon
See the Wiki untuk lebih detail tentang native didukung opsi baris perintah.

Ekspor Laporan Ekstensi-Opsi Baris Perintah

  • -export_report:

    • Description:

      • Menyimpan ke file yang ditentukan setelah loading yang diberikan sesi.

    • Currently Available

      • ASCII 1.0 Strict Compliant XHTML Files (.xhtml )
      • XML Files (.xml )
      • JSON Files (.json )

    • Future Milestone

      • Bootstrap HTML Files(.bootstrap.html )
      • PDF Documents (.pdf )
      • Google Documents (.doc )

  • -source_info:

    • Description:

      • Menentukan berikut rincian laporan:
        1. Title
        2. By
        3. For
        4. Scan Date
        5. Report Date
        6. Scan Ver
        7. Report Ver
        8. Description

    • Example:

      • -source_info "Vulnerability Report of MyApp.com;JordanGS;Lost Souls;August 15, 2016;August 18, 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;Lorem ipsum dolor sit amet, pri corpora ancillae adolescens in. Ea usu atomorum tincidunt, ne munere regione has. Ne sea summo tation, et sed nibh nostrum singulis. Nec causae viderer discere eu."

    • Validation:

      • Konten tidak dicentang, dapat memasukkan bidang kosong jika anda ingin, satu-satunya syarat adalah bahwa semua 8 item dalam daftar.

  • -alert_severity:

    • Description:

      • Menentukan mana peringatan severities akan dimasukkan dalam laporan:
        1. High
        2. Medium
        3. Low
        4. Informational

    • Input format:

      • Only accepts a string list with ';' delimiter

    • Input restriction:

      • Only accepts 't' and 'f' for each item in the list

    • Contoh:

      • -alert_severity "t;t;f;t"

    • Explanation:

      • Dalam contoh di atas, hanya lansiran tinggi, menengah dan Informasi yang akan disertakan dalam laporan yang dihasilkan.

    • Validation:

      • Validasi: konten divalidasi menjadi 't' atau 'f' dan semua 4 barang ada dalam daftar.

  • -alert_details:

    • Description:

      • Menentukan rincian peringatan mana yang akan disertakan dalam laporan:
        1. CWE ID
        2. WASC ID
        3. Description
        4. Other Info
        5. Solution
        6. Reference
        7. Request Header
        8. Response Header
        9. Request Body
        10. Response Body

    • Input format:

      • Only accepts a string list with ';' delimiter

    • Input restriction:

      • Hanya menerima 't' dan 'f' untuk setiap barang dalam daftar

    • Example:

      • -alert_details "t;t;t;t;t;t;f;f;f;f"

    • Explanation:

      • Dalam contoh di atas, hanya CWE ID, WASC ID, Deskripsi, Info Lainnya, Solusi dan Referensi Rincian Waspada akan dimasukkan dalam laporan yang dihasilkan.

    • Validation:

      • Content is validated to be either 't' or 'f' and that all 10 items are in the list.

  • Putting it all together! (Windows)

    1. Open a command prompt
    2. Menavigasi ke lokasi di mana zap.bat yang terletak
    3. execute the following:
      • zap.bat -export_report "/path/to/save/report.xhtml" -source_info "Vulnerability Report of MyApp.com;JordanGS;Lost Souls;August 15, 2016;August 18, 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;Lorem ipsum dolor sit amet, pri corpora ancillae adolescens in. Ea usu atomorum tincidunt, ne munere regione has. Ne sea summo tation, et sed nibh nostrum singulis. Nec causae viderer discere eu." -alert_severity "t;t;f;t" -alert_details "t;t;t;t;t;t;f;f;f;f" -session "/path/to/existing/session" -cmd

API Calls

Ekstensi dapat diakses dengan panggilan API dan memerlukan argumen berikut yang harus dilalui untuk menghasilkan laporan.

Pembatasan yang sama untuk Baris Perintah di atas.

Usage Example:


Map<String, String> map = new HashMap<String, String>();
String API_KEY = "ZAPROXY-PLUGIN"
String absolutePath = "/path/to/save/report.xhtml";
String extension = "xhtml";
String sourceDetails = "Vulnerability Report of MyApp.com;JordanGS;Lost Souls;" +
    "August 15, 2016;August 18, 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;" + 
    "Lorem ipsum dolor sit amet, pri corpora ancillae adolescens in. Ea " + 
    "usu atomorum tincidunt, ne munere regione has. Ne sea summo tation, " +
    "et sed nibh nostrum singulis. Nec causae viderer discere eu.";
String alertSeverity = "t;t;f;t";
String alertDetails = "t;t;t;t;t;t;f;f;f;f";

map.put("apikey", API_KEY);

// String representing the absolute path to save destination for the report
map.put("absolutePath", absolutePath)

// String representing the extension of the report
map.put("fileExtension", extension);

// String separated by ; for each entry
map.put("sourceDetails", sourceDetails);

// String separated by ; for each entry
map.put("alertSeverity", alertSeverity);

//String separated by ; for each entry
map.put("alertDetails", alertDetails);
ApiResponseElement val = (ApiResponseElement) clientApi.callApi("exportreport", "aksi", "menghasilkan", peta);
