Konsol script

Script Konsol memungkinkan anda untuk menulis skrip yang berjalan di dalam ZAP.
Hal ini terdiri dari:

  • Toolbar
  • Sebuah area di mana anda dapat menulis skrip anda
  • Sebuah area untuk debug dan pesan kesalahan
Anda dapat menjalankan 'Berdiri sendiri' script dari dalam tab ini menggunakan 'Run' tombol pada bagian alat tab.
Semua jenis lain dari script akan dijalankan ketika diaktifkan atau jika secara eksplisit dipanggil.

Untuk membuat script baru untuk memuat atau beralih melihat scrip Script 'pohon' tab.

Klik kanan di script daerah untuk tampilan dan pilihan editing.

Template dapat dilihat di konsol, namun tidak dapat diedit.

Auto-lengkap

Konsol mendukung bentuk terbatas dari auto-complete, dikendalikan oleh tombol pada toolbar.
When enabled a popup will be shown when you type the name of one of the defined parameters followed by a dot - this will show you all of the methods available for you to call. Memilih salah satu dari mereka akan menyisipkan metode dan parameter ke dalam naskah anda. If you type another dot immediately after the method call then you will be shown another prompt for the methods available to the return type, if any. Anda akan mampu untuk terus memperluas jenis kembali sampai anda mulai mengetik sesuatu yang lain.
