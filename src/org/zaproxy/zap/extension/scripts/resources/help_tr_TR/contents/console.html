Script Console Tab

The Scripts Console allows you to write scripts which run within ZAP.
It is made up of:

  • Bir araç çubuğu
  • Scriptlerinizi yazabileceğiniz bir alan
  • Hata ayıklama ve hata mesajları için bir alan
Araç çubuğu sekmesindeki 'Çalıştır' tuşuna basarak 'Yalnız çalışabilen' scriptleri çalıştırabilirsiniz.
All other types of scripts will be run when enabled or if explicitly invoked.

To create a new script of to load or switch scripts see the The Scripts 'tree' tab.

Görüntüleme ve düzenleme ayarları için script alanına sağ tıklayabilirsiniz.

Templates can be viewed in the console but cannot be edited.

Auto-complete

The console supports a limited form of auto-complete, controlled by a button on the toolbar.
Etkinleştirildiğinde, tanımlanan parametrelerin birinden sonra bir nokta koyduğunuzda bir açılır pencere görüntülenir -bu, aramaya hazır tüm yöntemleri gösterir. Selecting one of them will insert the method and parameters into your script. If you type another dot immediately after the method call then you will be shown another prompt for the methods available to the return type, if any. You will be able to keep expanding the types returned until you start typing something else.
