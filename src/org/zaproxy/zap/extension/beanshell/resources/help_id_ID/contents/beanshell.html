Bean Shell Console

BeanShell adalah sebuah shell Java interaktif yang bisa digunakan untuk mengeksekusi skrip BeanShell. Skrip ini adalah bentuk Java yang disederhanakan yang menggunakan banyak elemen dari sintaks Java, namun lebih sederhana format scripting. All Java code is also valid BeanShell code.
BeanShell integration in OWASP ZAP enables you to write scripts using the ZAP functions and data set. Ini bisa menjadi fitur yang sangat kuat untuk menganalisa aplikasi web.

Konsol BeanShell

Konsol dimulai dari menu alat, dan berisi layar split dimana bagian atas adalah konsol BeanShell interaktif dan separuh bawahnya adalah editor teks sederhana. Untuk naskah yang kompleks, kamu didorong untuk menggunakan editor Java. Skrip dapat dimuat, disimpan dan dievaluasi melalui editor. Ketika BeanShell memulai sejumlah objek dari ZAP yang tersedia untuk digunakan, dinamakan:

  • the Model singleton, through the object named model
  • the SiteTree tree of current sites through the sites object
  • an instance of 
    HttpSender
    through the pengirim objek
Notice that the BeanShell is loosely typed. Oleh karena itu, tidak perlu mendeklarasikan variabel sebelumnya menggunakannya - ini membuat skrip sedikit lebih ringkas dari pada Java biasa. Namun tentu saja jika anda mau menentukan tipe yang anda bisa.

Menggunakan Peta Situs

Mari kita mulai dengan sesuatu yang berguna dan khas: iterate melalui semua node situs dan periksa adanya sebuah berkas. A script has already been created that accomplishes this, choose Load and select the example.tree.bsh file. Before clicking Evaluate, first browse to a site through ZAP to populate the tree:

Sekarang klik evaluasi untuk mengeksekusi skrip yang ada pada penyunting. Jika tidak ada kesalahan, maka sekarang anda bisa memulai penggunaan objek yang didefinisikan dalam skrip dengan mengeluarkan perintah sebagai berikut: 

	t = Tree();
Yang membangun sebuah Pohon baru objek dan memberikan referensi untuk t. 
	t.find(sites.getRoot(), "index.html");
Panggil metode pencarian pada t, dimana mengambil SiteNode sebagai pendapat pertama dan sumber daya untuk menemukan yang kedua. In this case, the method will iterate through all the nodes in the tree, because we started at the root, and will find index.html files.

Tips

Use the unset(String) command to unset any declared variables, methods or objects. Hal ini berguna jika anda ingin mengganti metode deklarasi di saat namespace. Note that the command takes a String argument, not an Object, so to unset the t object we used above, it should be: unset(“t”); and not unset(t);
