Komut Satırı

Hızlı Başlangıç eklentisi aşağıdaki komut satırı seçeneklerini destekler:
     -hızlıurl Saldırı yapılacak olan hedef URL'sini belirtir.
     -hızlıçıkış XML raporu yazılacak dosya belirtilir. 'Satır içi' ve arka plan modlarında ayarlama yapılmazsa, rapor varsayılan çıktı akışına yazılır.
     -hızlıilerle Show ascii progress bars, if started with the -cmd flag.

Örnekler:

  • Start ZAP in 'inline' mode with ascii progress bars, attack the target application http://example.com/ and write the report to default output stream: 
    -cmd -quickurl http://example.com/ -quickprogress
  • Start ZAP with UI, attack the target application http://example.com/ and save the report to a file: 
    -quickurl http://example.com/ -quickout /path/to/report.xml
