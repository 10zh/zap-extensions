Hızlı Başlangıç Başlatma Seçenek Ekranı

Başlangıç Sayfası

This controls the page that is shown when you launch a browser to be proxied through ZAP.
3 seçenek var:
  • Varsayılan ZAP Sayfası
  • Boş Sayfa
  • URL (specify below)
Varsayılan ZAP sayfası başlangıçta çevrimiçi bir sayfadan okunur.
Çevrimiçi sayfanın talep edilmesini istemiyorsanız, diğer seçeneklerden birini seçebilirsiniz.

URL

The URL of the page that you would like browsers launched from ZAP to display.
