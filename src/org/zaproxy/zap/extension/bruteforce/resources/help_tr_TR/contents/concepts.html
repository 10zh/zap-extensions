Forced Browse

ZAP, zoraki tarama yöntemini kullanarak dizinleri ve dosyaları bulmaya çalışmanızı sağlar.
Çok sayıda dosya ve dizin adı içeren bir dizi dosya sağlanmaktadır.
ZAP, seçilen dosyada listelenen tüm dosyalara ve dizinlere doğrudan bağlantı sağlamaya çalışmak yerine doğrudan bunlara bağlantı bulmaya güvenmemektedir.

Forced Browse is configured using the Options Forced Browse screen.

This functionality is based on code from the OWASP DirBuster project.

Accessed via

     Forced Browse tab
     Sites tab 'Attack/Forced Browse site' right click menu item
     Sites tab 'Attack/Forced Browse directory' right click menu item
     Sites tab 'Attack/Forced Browse directory (and children)' right click menu item
