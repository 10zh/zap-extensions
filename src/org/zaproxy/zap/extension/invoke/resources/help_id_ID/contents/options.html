Pilihan Aplikasi layar

Layar ini memungkinkan anda untuk mengkonfigurasi aplikasi that can be invoked.
By default there are no applications available, you need to add all of the applications that you want to use.

Tampilan Nama

Nama yang akan digunakan untuk aplikasi ini didalam ZAP.

Perintah Penuh

Path lengkap dari aplikasi yang anda inginkan untuk memanggil.
Hanya satu perintah dapat ditentukan. Jika anda ingin menjalankan beberapa perintah sekaligus pergi, atau atur variabel lingkungan, lalu anda harus membuat skrip atau berkas batch dan kemudian panggil ini dari ZAP.

Parameter

Parameter yang harus dilalui ZAP ke perintah.
Tag berikut dapat digunakan untuk melewati parameter:
     %url% the full url, e.g. 'http://localhost/test?a=b'
     %site% the site, e.g. 'http://localhost:8080/'
     %host% the hostname, e.g. 'localhost'
     %port% the port, e.g. '80'
     %cookie% the first cookie field from the request header (if any)
     %postdata% the POST data sent, if any, with any newlines replaced with "\n"
So if a command accepted a URL using '-u' you would specify the parameters as: '-u %url%'

Capture Output

Jika dicek lalu perintah berjalan dan setiap output yang dihasilkan akan ditampilkan di Output tab.
Anda harus selalu menggunakan pilihan ini saat menjalankan skrip atau perintah yang tidak memiliki UI.
Anda juga bisa menggunakan pilihan ini untuk penyelesaian masalah - jika aplikasi tidak berjalan seperti yang diharapkan kemudian salin jalankan perintah dan mencobanya dari baris perintah cepat.

Keluaran ke Catatan

If checked then any output will be appended to a Catatan melekat ke yang relevan History tab rekam.
Perhatikan bahwa beberapa node dalam Sites tab tidak berhubungan Tab riwayat catatan - untuk node tersebut pilihan ini tidak akan memiliki efek.
