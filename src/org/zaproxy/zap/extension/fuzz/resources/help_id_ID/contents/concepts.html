Fuzzing

Fuzzing adalah teknik mengirimkan banyak yang tidak valid atau data yang tak terduga untuk target.

ZAP memungkinkan anda untuk fuzz apapun permintaan masih menggunakan:

  • Membangun di set muatan
  • Muatan yang didefinisikan oleh opsional add-ons
  • Custom scripts

Untuk mengakses Dialog fuzzer anda juga dapat:
  • Right click a request in one of the ZAP tabs (such as the History or Sites) and select “Attack / Fuzz…”
  • Highlight a string in the Request tab, right click it and select “Fuzz…”
  • Select the “Tools / Fuzz…” menu item and then select the request you want to fuzz

Payload Generators

Muatan Generator menghasilkan baku serangan yang telah mengirimkan hasil pengeditan. mengajukan untuk aplikasi target.

Mereka berhasil melalui Payloads dialog

Payload Processors

Muatan Prosesor dapat digunakan untuk mengubah muatan tertentu sebelum mereka diserahkan.

Mereka berhasil melalui Payload Processors dialog

Fuzz Location Processors

Fuzz Lokasi Prosesor dapat digunakan untuk mengubah semua muatan sebelum mereka diserahkan.

Mereka berhasil melalui Lokasi Prosesor dialog

Message Processors

Pesan Prosesor dapat mengakses dan mengubah pesan menjadi fuzzed, mengontrol fuzzing proses dan berinteraksi dengan ZAP UI.

They are managed via the Fuzzer dialog 'Message Processors' tab.

Fungsi ini berdasarkan dari kode proyek OWASP JBroFuzz dan termasuk berkas berkas dari proyek fuzzdb.
Perhatikan bahwa beberapa berkas fuzzdb telah ditinggalkan karena menyebabkan pemindai antivirus umum untuk menandai virus tersebut karena mengandung virus.
Anda dapat menggantinya ( dan meng-upgrade fuzzdb) dengan mendownload fuzzdb versi terbaru dan memperluasnya di 'fuzzer' library.
