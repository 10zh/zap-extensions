Fuzzer dialog

Hal ini memungkinkan anda untuk memilih fuzzers untuk digunakan ketika fuzzing a request.

Fuzz Locations tab

To configure the fuzzing:
  • Sorot sebuah string yang anda ingin bulu di Fuzz Lokasi tab
  • Click the 'Add...' button to launch the Payloads dialog
  • Tambahkan muatan yang ingin anda gunakan
  • Click on the 'Processors...' button to launch the Location Processors dialog (if required)
  • Click on the 'Start Fuzzer' button to start the fuzzing
  • The results will then be listed in the Fuzzer tab select them to see the full requests and responses.
You can also search for strings in the fuzz results using the 'Search tab'.

Options tab

Tab ini memungkinkan anda untuk mengkonfigurasi opsi-opsi yang akan digunakan ketika fuzzing.

Message Processors tab

Message Processors can access and change the messages being fuzzed, mengontrol fuzzing proses dan berinteraksi dengan ZAP UI.
Pesan prosesor yang tersedia tergantung pada jenis pesan yang fuzzed, add-on ini termasuk dalam default HTTP prosesor Pesan.

Add-ons juga dapat menentukan tambahan muatan generator.
