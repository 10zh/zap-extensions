Payload Processors dialog

This allows you to select the payload processors to use with specific payload generators.

Built in payload processors include:
  • Base64 Kod Çözme
  • Base64 Kodlama
  • Expand (to a minimum specified length)
  • JavaScript Escape
  • JavaScript Unescape
  • MD5 Hash
  • Postfix String
  • Prefix String
  • SHA-1 Hash
  • SHA-256 Hash
  • SHA-512 Hash
  • Trim
  • URL Kod Çözme
  • URL Kodlama
Ayrıca özel yük taşıma işlemcisi komut dosyalarına yazdırabilirsiniz; bunlar, size gereken yükün herhangi bir manipülasyonunu gerçekleştirebilir.

Add-ons can also define additional Payload Processors.
