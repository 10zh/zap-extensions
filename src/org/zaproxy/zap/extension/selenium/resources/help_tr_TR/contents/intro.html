Selenyum

Selenyum eklentisi WebDrivers'ı diğer eklentiler için web tarayıcılarını çağırıp uzaktan kontrol etmeyi sağlar. Aynı zamanda, kafasız bir Java tabanlı web tarayıcısı olan HtmlUnit web tarayıcısı da paketlenmiştir.

Eklenti, seçilen URL'yi desteklenen tarayıcılardan herhangi birinde açmanıza izin veren URL'leri (Geçmiş ve Siteler ağacında olduğu gibi) listeleyen ZAP panellerine sağ tıklama seçeneğini de ekliyor.

Desteklenen Tarayıcılar

Aşağıdaki web tarayıcıları desteklenmektedir:
İsim Kimlik Gereksinimler/Notlar
Chrome chrome Aşağıdaki sürümlerin çalıştığı bilinmektedir: 59 ve 60 (eski sürümler de işe yarayabilir). ChromeDriver gerektiriyor, eğer sistem yolunda yoksa, Seçenekler içinde ayarlanabilir. ChromeDriver ve onu nasıl elde edeceğiniz hakkında daha fazla bilgi için ChromeDriver websitesine bakın.
Firefox firefox Şu sürümlerin çalıştığı bilinmektedir: 45 (ESR), 46, 47.0.1, 54 ve 55 (daha eski sürümler de çalışabilir). Bazı sürümlerin çalışmadığı bilinmektedir, örneğin, 47.0. Daha yeni sürümler (≥ 48) geckodriver gerektirir, seçeneklerden ayarlanabilir. For more information on geckodriver and how to obtain it refer to the geckodriver website (see footer note for caveat when using geckodriver).
HtmlUnit htmlunit Birlikte verilen tarayıcının, hiçbir gereksinimi yoktur.
Internet Explorer ie The following version is known to work: 11 (older versions might work too). Requires IEDriverServer, if not on the system's PATH, it can be set in the options. For more information on IEDriverServer refer to the IEDriverServer website (see footer note for caveat when using Internet Explorer).
Opera opera Geçici olarak çalışmıyor.
PhantomJS phantomjs The following version is known to work: 2.1.1 (older versions might work too). Requires PhantomJS binary, if not on the system's PATH, it can be set in the options. For more information on PhantomJS and how to obtain it refer to the PhantomJS website (see footer note for caveat when using PhantomJS).
Safari safari  

Firefox, Chrome, Internet Explorer, Opera, PhantomJS ve Safari'yi kullanmak için, bunları sisteminize yüklemeniz gerekir. Tarayıcının kimliği, ZAP'ı komut satırı aracılığıyla yapılandırırken veya ZAP API'sini kullanırken tarayıcıyı seçmek için kullanılabilir (örneğin, AJAX Spider'ı bir veya başka bir tarayıcı kullanacak şekilde ayarlamak için).

Some of the requirements (e.g. WebDrivers) of the browsers can be configured in the Options Selenium screen.

Birlikte verilen WebDriverlar

ZAP WebDriverlar ile eklentiler sağlar, o eklentiler yüklendiğinde ZAP varsayılan olarak birlikte verilen WebDriverları kullanmayı deneyecektir. Bazı OS'lerde tarayıcıların bazıları için bir WebDriver'ı olmayabilir, bu durumda ZAP seçenekleri panelinde kullanılabilir bir WebDriver'ın mevcut olmadığını bildirecektir. The bundled WebDrivers can also be (re)set with the 'Bundled' button (for example, if another WebDriver was manually set). Not all browser versions are supported with the bundled WebDrivers, it's recommended that newer/latest versions of the browsers be used whenever possible.

Not: ZAP eklentileri ek tarayıcılar ekleyebilir.

Firefox/geckodriver Note: There's an issue (Bug 1103196) HTTPS sitelerinin sürümlerde kullanılmasını önler ≥ 48 and < 52.

PhantomJS Note: There's an issue (Issue #11342) that prevents sites at localhost, 127.0.0.1 and ::1 from being proxied through ZAP. Bir düzeltme bulunana kadar bu durumlarda kullanmamanız önerilir. Bazı eklentiler bu olduğunda uyarı iletisi göstermeyi seçebilir. As workaround one could define, in the hosts file, a domain name mapping to the local address and use that domain name instead.

İnternet Explorer Notu: Internet Explorer'ın tüm sürümleri kutudan çıkmaz, bkz IEDriverServer Web sitesi bunları yapılandırma hakkında daha fazla bilgi için.
