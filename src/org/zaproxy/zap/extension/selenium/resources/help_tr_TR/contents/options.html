Options Selenium screen

This screen allows you to setup the requirements of some of the WebDrivers.

Configuration option explanation

Yapılandırma Seçenekleri
Alan Ayrıntılar Varsayılan
WebDrivers
ChromeDriver This allows you to select the location of ChromeDriver. The path to the bundled WebDriver, if available.
geckodriver This allows you to select the location of geckodriver (Firefox driver). The path to the bundled WebDriver, if available.
IEDriverServer This allows you to select the location of IEDriverServer. The path to the bundled WebDriver, if available.
Binaries
Firefox Bu Firefox ikilisinin konumunu seçmenize izin verir (örneğin, sistem varsayılanından farklı bir sürüm kullanmak için). (Yok)
PhantomJS Bu PhantomJS ikilisinin konumunu seçmenize izin verir. (Yok)
Not: Yukarıdaki konumların (ikili dosyalar ve Web Sürücüleri) Java sistem özelliklerini kullanarak da ayarlanması mümkündür, bu durumda yukarıdaki seçenekler geçersiz olur ve bunun yerine gösterilen değerler gelir.

Paketlenmiş WebDrivers hakkında daha fazla detay genel bakış sayfasında bulunabilir.
