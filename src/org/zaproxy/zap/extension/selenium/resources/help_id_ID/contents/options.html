Options Selenium screen

This screen allows you to setup the requirements of some of the WebDrivers.

Configuration option explanation

Configuration Options
Field Details Default
WebDrivers
ChromeDriver Ini memungkinkan anda untuk memilih lokasi dari ChromeDriver. The path to the bundled WebDriver, if available.
geckodriver Ini memungkinkan anda untuk memilih lokasi dari geckodriver (driver Firefox). The path to the bundled WebDriver, if available.
IEDriverServer Ini memungkinkan anda untuk memilih lokasi dari IEDriverServer. The path to the bundled WebDriver, if available.
Binari
Firefox Ini memungkinkan anda untuk memilih lokasi dari biner Firefox (contohnya, untuk menggunakan versi selain dari sistem default). (None)
PhantomJS This allows you to select the location of PhantomJS binary. (None)
Note: Ini juga memungkinkan untuk mengatur lokasi-lokasi di atas (binari dan WebDrivers) menggunakan properti sistem Java, sehingga opsi di atas akan ditimpa dan nilai tersebut ditampilkan.

Rincian selengkapnya tentang bundel WebDrivers bisa ditemukan pada halaman ikhtisar.
