Open API Specification Support

This add-on allows you to spider and import OpenAPI (Swagger) definitions (versions 1.2 and 2.0).

Add-on akan mendeteksi secara otomatis setiap definisi API Terbuka dan menggerakkannya selama berada di dalam jangkauan.

UI

2 item menu ditambahkan pada menu perkataan:
  • Import an Open API definition from the local file system
  • Import an Open API definition from a URL
Dialog URL Impor mendukung bidang penambahan opsional yang memungkinkan anda untuk menolak Host (dan port) di definisi dengan satu alternatif yang anda suplai.
Pilihan ini tidak disediakan untuk impor berkas karena mudah untuk merubah berkas lokal untuk menggunakan host yang berbeda.

API

Operasi berikut ini ditambahkan ke API:
  • ACTION importFile (file)
  • ACTION importUrl (url, hostOverride)
Definisi akan diimpor secara serentak dan setiap peringatan akan dikembalikan.

Command Line

The following Command Line options are added:
  • -openapifile <filename> : Import an Open API definition from the specified file name
  • -openapiurl <url> : Import an Open API definition from the specified URL
The definitions will be imported synchronously and any warnings will be displayed on the command line.

User Specified Values

Default values are used when importing Open API definitions.
Ini dapat diganti menggunakan add-on Bentuk Handler yang memungkinkan anda untuk menentukan nilai anda sendiri.
In most cases these will be simple values (like strings and integers) but in some cases you may need to specify structured values, e.g. 
{ "id": 0, "name": "Freda" }
