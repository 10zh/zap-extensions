Open API Specification Support

This add-on allows you to spider and import OpenAPI (Swagger) definitions (versions 1.2 and 2.0).

The add-on will automatically detect any Open API definitions and spider them as long as they are in scope.

Kullanıcı Arayüzü

Araçlar menüsüne 2 menü öğresi eklenir:
  • Import an Open API definition from the local file system
  • Import an Open API definition from a URL
URL İçe Aktarma iletişim kutusu, tanımdaki Ana Bilgisayarı (ve bağlantı noktasını) sizin sağladığınız alternatif bir sunucu ile geçersiz kılmanıza izin veren isteğe bağlı bir ek alanı destekler.
This option is not provided for the file import as it's easy to change a local file to use a different host.

API

API'ya aşağıdaki işlemler eklenir:
  • ACTION importFile (file)
  • ACTION importUrl (url, hostOverride)
The definitions will be imported synchronously and any warnings will be returned.

Komut Satırı

Aşağıdaki Komut Satırı seçenekleri eklenir:
  • -openapifile <filename> : Import an Open API definition from the specified file name
  • -openapiurl <url> : Import an Open API definition from the specified URL
Tanımlar eşzamanlı olarak içe aktarılacak ve uyarılar komut satırında görüntülenecektir.

Kullanıcı Belirli Değerler

Default values are used when importing Open API definitions.
These can be overridden using the Form Handler add-on which allows you to specify your own values.
In most cases these will be simple values (like strings and integers) but in some cases you may need to specify structured values, e.g. 
{ "id": 0, "name": "Freda" }
