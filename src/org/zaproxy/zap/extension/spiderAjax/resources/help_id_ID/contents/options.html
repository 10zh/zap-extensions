Pilihan layar Spider AJAX

Layar ini memungkinkan anda untuk mengkonfigurasi Laba-laba AJAX pilihan. Spider AJAX adalah add-on untuk sebuah crawler yang disebut Crawljax. Add-on mendirikan sebuah proxy lokal di ZAP untuk berbicara dengan Crawljax. Spider AJAX memungkinkan anda untuk menjelajahi aplikasi web yang tertulis di AJAX jauh lebih mendalam daripada Spider yang asli. Gunakan Spider AJAX jika anda mungkin memiliki aplikasi web yang tertulis dalam AJAX. Anda juga harus menggunakan Spider yang asli serta untuk liputan lengkap aplikasi web (misalnya, untuk menutupi komentar HTML).

Penjelasan pilihan konfigurasi



Pilihan Konfigurasi
Bidang Rincian Default
Browser Spider AJAX bergantung pada browser eksternal untuk menjelajahi situs yang ditargetkan. Anda bisa menentukan mana yang ingin anda gunakan. Untuk rincian yang lebih lanjut tentang peramban yang didukung lihat halaman bantuan add-on "Selenium". Firefox
Jumlah jendela browser yang terbuka Anda dapat mengonfigurasikan jumlah jendela yang akan digunakan oleh AJAX Spider. Semakin banyak jendela, semakin cepat prosesnya. 1
Kedalaman penjelajahan maksimum Kedalaman maksimum yang dapat dijangkau penjelajah. Nol berarti kedalaman yang tak terbatas. 10
Penjelajah maksimum negara bagian Jumlah maksimum negara bagian yang penjelajahnya harus dijelajahi. Nol berarti negara yang penjelajah tidak terbatas. 0 (tidak terbatas)
Durasi maksimum Waktu maksimum yang diizinkan penjelajah untuk menjalankannya. Nol berarti waktu yang berjalan tidak terbatas. 60 menit
Waktu tunggu acara Waktu untuk menunggu setelah sebuah acara sisi klien yang dipecat. 1000 ms
Muat ulang waktu tunggu Waktu untuk menunggu setelah URL dimuat. 1000 ms
Klik elemen sekali Ketika diaktifkan, penjelajah mencoba untuk berinteraksi dengan setiap elemen (misalnya, dengan mengeklik) hanya sekali. If this is not set, the crawler will attempt to click multiple times. Unsetting this option is more rigorous but may take considerably more time. True
Use random values in form fields When enabled, inserts random values into form fields. Otherwise, it uses empty values. True
Click default elements only When enabled, only elements "a", "button" and "input" will be clicked during crawl. Otherwise, it uses the table below to determine which elements will be clicked. For more in depth analysis, disable this and configure the clickable elements in the table. True
Select elements to click during crawl (table) The list of elements to crawl. This table only applies when "click default elements only" is not enabled. Use "enable all" for a more in depth analysis, though it may take somewhat longer. All enabled
