WebSocket specific options

Bu ekran şu WebSocket seçeneklerini yapılandırmanıza olanak verir:

Tümünü ilet

Eğer WebSockets iletişimi ile ilgilenmiyorsanız fakat mesajlarının ZAP vasıtasıyla geri & ileri gönderilmesine izin vermek istiyorsanız, bu seçeneği etkinleştirmelisiniz. Sonuç olarak oturum veritabanında hiçbir mesaj saklanmayacaktır. Ayrıca WebSocket sekmesinde hiçbir ileti görünmez. Bu ayar, performans kritik WebSocket bağlantılarıyla uğraşmanız gerekiyorsa ve neyin gönderildiğiyle ilgilenmiyorsanız yararlı olabilir.

WebSocket'e özel belirli bir kanalı yok saymak isterseniz Session Properties.

Break on all

In the Break toolbar, there are two buttons:
  •   Set break on all requests
  •   Set break on all responses
Bu seçeneği etkinleştirdiğinizde, giden (sırasıyla gelen) WebSocket mesajları da bu kontrol düğmeleri tarafından yakalanır.

Varsayılanlar tarafından bu ayar kapatıldı. Bu durumda yalnızca HTTP iletileri yakalanır ancak WebSocket iletişiminde açık kesme noktaları ekleyebilirsiniz WebSocket tab.

Break on Ping & Pong messages

PING ve PONG mesajları, WebSocket sunucuları tarafından geçerli bağlantının sağlığını saptamak için sık sık kullanılır. Eğer gelen bir yakalarsanız PING message, the browser is not able to return a PONG message immediately, resulting in a bad latency value.

If this option is disabled no PING or PONG message is caught when you:
  • set a breakpoint on all requests or all responses, or
  • step through to the next request or response and a PING or PONG message arrives
Bu ayar açıkça belirlenen kesme noktalarını etkilemez PING or PONG messages. You can view such explicit breakpoints in the Breakpoints tab.

Remove Sec-WebSocket-Extensions header

Allows to remove the HTTP header Sec-WebSocket-Extensions from handshake messages, so no transformations are done to the WebSocket messages sent/received.
Bu seçenek, istemci veya test edilen sunucu gerektirmediği sürece daima etkinleştirilmelidir. Uzantılar kullanılırken WebSocket iletileri ZAP tarafından doğru şekilde işlenemeyebilir.
